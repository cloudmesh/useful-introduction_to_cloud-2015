Lanaguage Features¶

Print¶

>>> print "Hello Cloud"
Hello Cloud

Comments¶

Simple comment¶

>>> # this is a comment

Multiline Comment with “¶

"""
this is a multiline comment
"""

Multiline Comment with ‘¶

'''
this is a multiline comment
'''

Variables¶

>>> flavor = "tiny"
>>> print flavor
tiny

Changing the variable Value¶

>>> flavor = "large"
>>> print flavor
large

List¶

>>> list = [1, 2, 3, 4, 5, 6]
>>> print list[0]
1
>>> print list[len(list)-1]
6
>>> print list[2:5]
[3, 4, 5]

For Loop¶

>>> list = [1, 2, 3, 4, 5, 6]
>>> for element in list:
...   print element
... 
1
2
3
4
5
6

>>> flavors = ['tiny', 'medium', 'large']
>>> for flavor in flavors:
...   print flavor
... 
tiny
medium
large

If Condition¶

>>> flavor = "tiny"
>>> if flavor == "tiny":
...    print("vanilla has a tiny flavor")
... elif flavor == "large":
...    print("large flavor")
... else:
...    print("this flavor I do not like")
... 
vanilla has a tiny flavor

Arithmetic Operators¶

>>> list = [1, 2, 3, 4, 5, 6]
>>> print sum(list)
21
>>> print min(list)
1
>>> print max(list)
6
>>> print sum(list)/len(list)
3
>>> print sum(list)/float(len(list))
3.5

Functions¶

Function to do calculations¶

>>> def f(x,y):
...   return x+y+ y*y + x*x
... 
>>> print f(1,2)
8
>>> print f(4,6)
62

Functions for String Manipulation¶

>>> def name(firstname, lastname):
...   return "%s %s" % (firstname, lastname)
... 
>>> def reversename(firstname, lastname):
...   return "%s, %s" % (lastname, firstname)
... 
>>> print name('Gregor', 'von Laszewski')
Gregor von Laszewski
>>> print reversename('Gregor', 'von Laszewski')
von Laszewski, Gregor

Excersises¶

her is the task i like you to do to reinforce the learning experience once you are done with the tutorial.

  1. Write a program that uses loops over both x and y coordinates while x is in 1,2,3,4,5 and y is in 5,4,3,2,1 and prints the x and y coordinate
  2. Write a program that sums up all values in x and y
  3. Write a program just like the first task but does not print values where x is equal to 2 and y is equal to 4
  4. Write a function that takes in a word and returns it in reverse order
  5. Provide a program that uses dicts
  6. Read up on classes and write a program that uses classes.
  7. We will create an icecream machine that produces icecream in with tiny flavor, medium flavor and large flavor.
  8. In addition the icecream cone will be wrapped into some paper that has an image on it. Images will be Penguin, Apple, Emperor, King
